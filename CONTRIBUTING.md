# Contributing to Node-RED Process Link

Thank you for your interest in contributing to this project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/node-red-contrib-processlink.git`
3. Create a branch: `git checkout -b feature/your-feature-name`

## Development Setup

1. Install Node-RED globally or use an existing installation
2. Link this package for development:

```bash
cd node-red-contrib-processlink
npm link

cd ~/.node-red
npm link @processlink/node-red-contrib-processlink
```

3. Restart Node-RED to see your changes

## Code Style

- Use 2 spaces for indentation
- Use semicolons
- Follow existing code patterns in the repository

## Submitting Changes

1. Commit your changes with a clear message
2. Push to your fork
3. Open a Pull Request against the `main` branch
4. Describe your changes and link any related issues

## Reporting Issues

- Use the GitHub issue tracker
- Include Node-RED version, Node.js version, and OS
- Provide steps to reproduce the issue
- Include any error messages or logs
